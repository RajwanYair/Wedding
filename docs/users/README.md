## Diátaxis split (target)

User docs grow toward the four Diátaxis quadrants:

| Quadrant | Purpose | Status |
| --- | --- | --- |
| **Tutorial** | Learning-oriented walk-through | ⏳ planned (`tutorial-first-event.md`) |
| **How-to** | Goal-oriented recipes | ⏳ planned (`how-to/` folder) |
| **Reference** | Information-oriented (data shapes, fields) | ⏳ planned (`reference/`) |
| **Explanation** | Understanding-oriented (concepts, decisions) | ✅ ADRs serve this role today |

The reorganisation plan lives in [ADR-021](../adr/021-diataxis-docs-reorganization.md).
